The guides are task-oriented: each walks one operator job from start to finish.
Install the rib into a running Keelson first, then author the Minds, rooms, and
lenses that fill the Chamber surface.

  <LinkCard title="Install Chamber into Keelson" href="install/" description="Add the rib, choose which ribs activate, and open the Chamber surface." />
  <LinkCard title="Author a mind" href="author-a-mind/" description="Create a persistent agent with genesis, enter it for a direct chat, and retire it." />
  <LinkCard title="Run a room" href="run-a-room/" description="Convene Minds, pick a strategy, and steer the live conversation." />
  <LinkCard title="Author a lens" href="author-a-lens/" description="Render an agent-authored canvas board on a subject and keep it current." />
